Social Solar: User Testing

Objective: Evaluate usability and improve user experience for Social Solar, a platform for eco-conscious energy choices.

Methodology: Conducted moderated user tests focusing on the platform's onboarding process and interface elements. Provided five main recommendations to enhance user understanding and ease of use.

Outcome: Delivered actionable insights that increased sign-up probability by optimizing the information layout and improving the registration flow.

Alexander Street Press: UX Redesign

Objective: Improve the accessibility and content discoverability for Alexander Street's landing page.

Methodology: User interviews and observations on similar platforms. Suggested structural changes, including a refined content map and enhanced research tools, for better engagement.

Outcome: Proposed features such as curated playlists and a simplified sign-up process to align the landing page with users' expectations.

SAI Global: Content Strategy

Objective: Help SAI Global streamline brand messaging and improve content engagement.

Methodology: Competitive analysis, content audit, and voice tone analysis. Recommendations included organizing web real estate for clarity and reducing information overload.

Outcome: Created a content model that aligned brand tone with user needs, enhancing clarity and user trust in SAI Global's services.
